# 💰 MoneyTracker - Smart Financial Management

A premium, all-in-one financial tracking platform designed to help you take control of your financial future. Featuring a modern, high-performance dark theme with absolute clarity on your spending, investments, and net worth.

---

## ✨ Key Features

### 🏢 **Public Experience**
- **Dynamic Landing Page:** High-conversion hero section with interactive feature highlights.
- **Smart FAQ:** Accordion-style help center for quick user guidance.
- **Subscriptions:** Transparent, three-tier pricing model for various financial needs.
- **Contact & Support:** Integrated contact experience for seamless communication.

### 🔐 **Secure Authentication**
- **Premium Auth Flow:** Glassmorphism-inspired Login and Signup experience.
- **Onboarding Funnel:** Plan selection integrated directly into the signup process.
- **Google Integration:** Built-in support for social authentication via Google.

### 📊 **Power Dashboard**
- **Unified Overview:** Real-time stats for Total Balance, Income, Expenses, and Savings.
- **Accounts:** Centralized management for Banks, Cards, and Investment accounts.
- **Smart Transactions:** Deep-dive into spending with advanced filters and category tracking.
- **Budgeting Suite:** Visual progress bars and automated "On Track" vs. "Over Budget" alerting.
- **Investment Portfolio:** Track Assets, Stocks, and Crypto with 24h performance monitoring.
- **Customizable Settings:** Manage profiles, security, and subscription tiers.

## 🛠️ Tech Stack

- **Frontend:** Angular 17+ (Signals, Standalone Components)
- **Styling:** SCSS (Custom Design System, CSS Variables, Glassmorphism)
- **Icons:** Modern Emoji & Custom Assets
- **Layout:** Fully Responsive Sidebar & Top Navigation

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [Angular CLI](https://angular.io/cli)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/Money_Tracker.git
   cd Money_Tracker
   ```

2. **Frontend Setup:**
   ```bash
   cd frontend
   npm install
   ```

3. **Start the Development Server:**
   ```bash
   npm start
   ```
   Open your browser and navigate to `http://localhost:4200`

## 🎨 Design Philosophy

MoneyTracker is built on the **Premium Dark** aesthetic, utilizing:
- **Glassmorphism:** Layered card designs with `backdrop-filter` blur effects.
- **Dynamic Gradients:** Smooth indigo and purple transitions for a modern feel.
- **Micro-interactions:** Hover effects and state changes to keep the UI feeling "alive."
- **Typography:** Weighted headers and high-contrast text for maximum readability.

## 🛡️ Security

We prioritize your financial privacy.
- 256-bit AES encryption for all sensitive data.
- No direct storage of bank credentials.
- Multi-factor authentication support.
